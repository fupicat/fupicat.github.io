Curassaum

Instruções

Controles: Mouse ou teclado

No menu, clique no botão “Mouse / Touchscreen” para jogar o modo single-player.

No menu, clique no botão “Keyboard Co-op” para jogar o modo multiplayer local.

Objetivo

A célula branca deve encostar no invasor branco, e a célula preta deve encostar no invasor preto. Se uma célula encostar em um invasor da cor errada, ou os invasores chegarem ao coração, você perde.

Single-player

Mova o mouse para controlar as células protetoras.

Multiplayer local

Use WASD para mover a célula preta, e IJKL para mover a célula branca.

Sobre

Este jogo foi criado para a game jam Ludum Dare 46.

Fiquei em 716º lugar de 4959 jogos participantes! Estou começando a achar que não fui tão mal assim.

Lançado em 19 de abril de 2020.
